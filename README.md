This is a fullstack personal project with Nextjs 13 to study purposes.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech stack

- Nextjs 13;
- React 18;
- PostgreSQL (Railway);
- Prisma;
- Tailwind CSS;
- Typescript.
